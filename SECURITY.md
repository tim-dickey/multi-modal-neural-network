# Security Policy

## Supported Versions

The following table lists which versions of the multi-modal neural network currently receive security updates. We provide security updates for the latest release. Older versions do not receive security updates.

If you are using a version marked as unsupported, we strongly recommend upgrading to a supported version to ensure you receive the latest security patches.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.0   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

To report a security vulnerability, please use the GitHub Security Advisories feature for this repository:
[https://github.com/tim-dickey/multi-modal-neural-network/security/advisories](https://github.com/tim-dickey/multi-modal-neural-network/security/advisories)

We aim to respond to all vulnerability reports within **5 business days**.

- If your report is accepted, we will work with you to coordinate a fix and disclosure.
- If your report is declined, we will provide a clear explanation of our reasoning.
